Voice Recorder Redesign Concepts

Four alternative UI designs for the IELTS speaking practice recorder. Click each state tab to preview how the recorder looks in that mode.

1

Floating Pill

A floating capsule anchored to the bottom of the screen. Everything is reachable with one thumb. Inspired by WhatsApp/Telegram voice messages but more polished. The pill morphs between states, keeping the user focused on a single element.

2

Bottom Sheet

A frosted-glass bottom sheet that expands upward. Features a circular progress ring showing elapsed time as a percentage of the 4-minute limit. The sheet has a satisfying height transition between states, creating a sense of progression.

3

Card Stack

A clean card that transforms between states with clear visual hierarchy. No pause button -- a simpler two-step flow (record then done). Features a full-width immersive waveform during recording and clear dual-action buttons in review mode.

4

Inline Minimal

Ultra-minimal design that lives inline with the content. No overlays, no sheets. Uses animated dots instead of traditional waveform bars. Review mode becomes a modern mini-player. Focuses on being non-intrusive and blending seamlessly with the page.
